### 리액트의 이벤트 시스템

1. 이벤트를 사용할 때 주의사항
- 이벤트 이름은 카멜 케이스로 작성(ex: onclick -> onClick, onkeyup -> onKeyUp)
- 이벤트에 실행할 자바스크립트 코드를 전달하는 것이 아니라, 함수 형태의 값을 전달한다.
- DOM 요소에만 이벤트를 설정할 수 있다.(컴포넌트 자체에는 불가능)

2. 이벤트종류
- Clipboard
- Composition
- Keyboard
- Focus
- Form
- Mouse
- Selection
- Touch
- UI
- Wheel
- Media
- Image
- Animation
- Transition
